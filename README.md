## 🚀 What I Actually Build at Work

> At **Surepass**, I work on data pipelines that process thousands of API calls daily —
> vehicle RC verification, GST lookups, PAN checks, MCA company data.
> Here's what that looks like in practice:

```
📥  Ingest     →  CSV input files with 10,000+ rows
🔗  Fetch      →  Concurrent API calls (ThreadPoolExecutor / multiprocessing)  
🧹  Parse      →  Flatten nested JSON responses → structured CSV
🔍  Validate   →  Chassis matching, data masking, field-level logic
📤  Export     →  Clean, annotated output CSVs with comment columns
📊  Visualize  →  Power BI dashboards + Excel reporting for stakeholders
```

**Domains I've worked on:**
`Vehicle RC Data` · `GST Verification` · `PAN Lookup` · `MCA Company Data` · `Land Records`
