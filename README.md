## 🚀 What I Actually Build

```
INPUT   ──▶  CSV with 10,000+ rows  (RC numbers, GST INs, PAN IDs, CIN codes)
               │
FETCH   ──▶  Concurrent REST API calls via ThreadPoolExecutor
               │  (Surepass identity verification endpoints)
               │
PARSE   ──▶  ujson.loads() → flatten nested JSON → structured DataFrame
               │
VERIFY  ──▶  Chassis match · data masking · field-level validation logic
               │
EXPORT  ──▶  Annotated CSV with comment columns + error flags
               │
REPORT  ──▶  Power BI dashboard + Excel summary for business stakeholders
```

**Domains:** `Vehicle RC` · `GST` · `PAN` · `MCA Company` · `Land Records`
